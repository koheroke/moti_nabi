# Project Context

This is a Vue 3 + TypeScript + Vite project.

## Path Aliases

- `@` = `src`
- `@components` = `src/components`
- `@features` = `src/features`
- `@pages` = `src/pages`
- `@stores` = `src/stores`
- `@lib` = `src/lib`

## Rules

- Always use alias imports instead of relative paths
- Example:
  - `@features/user/types/profile`
  - `@components/Header.vue`
